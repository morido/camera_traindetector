Request_Next_Image

function Request_Next_Image return Number_Of_Chunks

Purpose

Ask the server to internally grab a new image from the camera and prepare it for transmission (i.e. crop it to the given size)

Parameters

None.

Returns

The number of chunks this image will be split into.

Exceptions

IMAGE_DIMENSION_ERRORThe dimensions of the requested imaged do not make sense (i.e. the lower right corner is left of/higher up than its upper left counterpart)
COMMUNICATION_ERRORThe server did not answer the request correctly or transmitted a specific errormessage
SOCKETCOMM.CONNECTION_ERRORno data was received within a given timeout
